# 📊 Sales Distribution Dashboard | Microsoft Excel & Power Query

## 📌 Project Overview

The **Sales Distribution Dashboard** is an end-to-end data analytics project developed using **Microsoft Excel** and **Power Query**. The objective of this project was to transform raw sales data into an interactive dashboard that enables stakeholders to monitor sales performance, identify business trends, and support data-driven decision-making.

The project demonstrates the complete data analysis lifecycle, including data extraction, transformation, cleaning, analysis, visualization, and dashboard development.

---

# 🎯 Project Objectives

The primary objectives of this project were to:

- Clean and transform raw sales data into a structured format.
- Analyze sales performance across different regions, products, customers, and time periods.
- Develop an interactive Excel dashboard for business reporting.
- Present meaningful insights through clear visualizations and KPIs.
- Demonstrate practical skills in Excel-based Business Intelligence.

# 📥 Data Import

The raw sales dataset was imported into Microsoft Excel using the **Get Data** feature and loaded into **Power Query** for preprocessing.

This approach ensured that all data transformations remained dynamic and reproducible without modifying the original dataset.

---

# 🧹 Data Cleaning & Transformation

Data preparation was performed using **Power Query** to improve data quality before analysis.

The following transformations were applied:

- Removed duplicate records
- Corrected inconsistent text formatting
- Standardized column values
- Changed appropriate data types
- Cleaned unnecessary spaces and formatting issues
- Prepared the dataset for analysis
- Loaded the cleaned dataset back into Microsoft Excel

The cleaned dataset served as the foundation for all subsequent analysis and dashboard visualizations.

---

# 📈 Data Analysis

After data preparation, the cleaned dataset was analyzed using Excel's analytical features.

The analysis focused on:

- Monthly sales performance
- Weekly sales trends
- Regional sales distribution
- Country-wise sales analysis
- Top-performing products
- Highest-value customers
- Sales performance against monthly targets

Pivot Tables and Pivot Charts were used to summarize and organize the data efficiently before designing the dashboard.

---

# 📊 Dashboard Features

The dashboard provides an interactive overview of sales performance through the following components:

### Key Performance Indicators (KPIs)

- Total Sales

### Interactive Filters

- Month Slicer
- Region Slicer

### Visualizations

- Monthly Target vs Actual Sales
- Weekly Sales Trend
- Regional Sales Distribution (Donut Chart)
- Country-wise Sales Map
- Top 10 Products
- Top 3 Customers

The interactive slicers allow users to dynamically filter the dashboard and explore sales performance across different regions and months.

# 💼 Business Questions Addressed

This dashboard helps answer several important business questions, including:

- What is the overall sales performance?
- How does actual sales compare with monthly targets?
- Which regions contribute the highest sales?
- Which customers generate the most revenue?
- Which products are the best-performing?
- How do sales fluctuate throughout the weeks?
- Which countries contribute to overall sales?

---

# 📌 Key Insights

The analysis highlights several business insights:

- Sales performance remained relatively consistent throughout the reporting period.
- A small group of customers contributed significantly to overall revenue.
- Product sales were concentrated among a limited number of high-performing products.
- Regional sales distribution was balanced, with slight variations across different regions.
- Weekly sales showed moderate fluctuations while maintaining overall stability.
- Interactive filtering enables quick identification of sales trends across regions and months.
