# 🌸 FNP Sales Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Sales Analysis Dashboard** developed in Microsoft Excel to analyze sales performance for Ferns N Petals (FNP).

The dashboard provides insights into revenue trends, product performance, customer behavior, city-wise orders, and occasion-based sales using interactive visualizations and slicers.

---

## 🎯 Business Objective

The primary objective of this dashboard is to help business stakeholders:

- Monitor overall sales performance
- Identify high-performing products
- Analyze seasonal revenue trends
- Understand customer purchasing patterns
- Track occasion-wise sales
- Support inventory and marketing decisions

---

## 📈 Key Insights

- Anniversary generates the highest revenue.
- Colors is the highest-performing product category.
- Revenue peaks during February, August, and November.
- Magnum Set is the top-selling product.
- Imphal records the highest number of orders.
- Saturday generates the highest weekly revenue.
- Average delivery time is approximately **5.53 days**.

---

## 💼 Business Recommendations

- Increase inventory before peak sales months.
- Promote best-selling products through bundle offers.
- Focus marketing campaigns around high-revenue occasions.
- Expand logistics support in top-performing cities.
- Introduce weekend promotional campaigns.
- Monitor delivery time to improve customer satisfaction.
